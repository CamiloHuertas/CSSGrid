Capitulo 2 - Grid Explicita

fr: es una nueva medida, significa el espacio sobrante

display: grid; Declara estado grid (se le aplica al padre)

grid-template-columns: Permite definir numero de columnas (su valor afecta el width

grid-template-rows: permite definir numero de filas (su valor afecta el height)

gap: determina un espacio entre grid items, su primer valor es arriba y abajo y el segudno es derecha e izquierda

Metodos mas avanzados

  • repeat(): este se usa en grid-template-columns y rows, recibe 2 valores, el primero es cuantas columns o rows queremos y el segundo es cuanto queremos que mida cada una

Capitulo 3 -Posicionamiento con grid lines

Esta la propiedad span que indica desde su " posicion actual hasta", pero hasta donde le indicemos mide en grid cells, no en grid lines

    Todas dicen de que linea a que linea quiero que se ocupe
  • Grid-row-start:
  • Grid-column-start:
  • Grid-row-end:
  • Grid-colum-end:

  • grid-row: esto acorta grid row start y end, recibe como valor grid-row: start / end
  • grid-column: esto acorta grid column start y end, recibe como valor grid-column: start / end

    Tambien le podemos decir que ocupe mas espacio de la grid si ponemos mas valor al end que 1 grid cell

  • grid-area: esto acorta grid-row y grid-column, primero va el valor del grid-row y despues el del grid-column

Capitulo 5- Posicionamiento con grid Areas

Se utiliza el grid-template-areas para ponerle nombres a areas de nuestra grid, se pone en el css como queremos que vaya nuestro html con nombre de las clases entre comillas

Para que esto funcione se le debe poner al elemento una propiedad grid-area que como valor tenga el mismo valor de su clase

Tambien si queremos dejar algo en blanco sin que lo ocupe un elemento, se reemplaza esa posicion por un .

Capitulo 7 - Flujo de la grid (grid flow)

Grid Auto flow

Esta tiene como propiedad estandar row. Lo que hace es cambiar la orientacion de la grid, como un flex direction de flexbox

Grid-auto-rows:

Esta propiedad afecta a las rows de la grid implicita, afecta especificamente su altura

Grid-auto-columns:

Esta propiedad afecta a las columns de la grid implicita, afecta especificamente su anchura

Capitulo 8 - Flujo denso de la grid

Si se le pide a un cell grid que ocupe mas cell grids de las que puede permitir la grid, lo va a mandar hacia abajo, no creara otra columna

Si despues de mover una grid cell o expandirla y quedan espacios sin rellenar, añadiendo la palabra "dense" a la propiedad grid auto flow se arregla, moviendo las cells debajo de este espacio a ese lugar

Capitulo 10 - Ordenamiento de grid items

La propiedad order tiene como valor definido 0, este se puede cambiar a negativos o positivos

justify-items: sirve para posicionar los elementos hijo, tiene como valor definido stretch
tiene los mismos valores de flexbox sin el flex, center, end, start

align-items: tambien tiene los valores de flexbox

Si queremos posicionar un elemento especifico se usa justify-self y align-self

Capitulo 11 - Alineacion de grid tracks

Justify-content: mueve todo el track, no afecta a los elementos hijo


Tiene las mismas propiedades del Justify-content de flexbox

align-content: mueve todo el track en el eje Y

Capitulo 12 - Tamaños max y min de grid tracks

Para declarar cuanto ocupa cada columna se puede utilizar la funcion minmax(), recibe 2 valores, el primero el valor minimo de cada columna y el max el maximo de cada columna

Como valor del min se puede poner min-content, que significa que se adaptara al tamaño minimo que tenga el contenido

min-content: la columna se adapta al contenido minimo que tenga

max-content: la columna se adapta al contenido maximo que tenga
